# Odin Recursion

A small JavaScript project built as part of **The Odin Project** curriculum to practice **recursion**, **problem decomposition**, and **algorithmic thinking**.  
The project focuses on implementing the **Fibonacci sequence** and **Merge Sort**, both iteratively and recursively, to develop a deeper understanding of how recursive solutions work under the hood.
